uploadcare-csharp

[![Build Status](https://travis-ci.org/okolobaxa/uploadcare-csharp.svg)](https://travis-ci.org/okolobaxa/uploadcare-csharp)

This is a Java library for Uploadcare.

Supported features:

- Part of file and project API v0.3
- CDN path builder
